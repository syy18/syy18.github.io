端午记

2026年6月19日 · 随笔

今日端午。

晨起，窗外雨声淅沥，空气里浮动着粽叶的清香。节令到了，该有的仪式感还是要有的。虽然医生说只能浅尝，但浅尝也是尝，图的是个过节的心情。

泡了一壶龙井，坐在电脑前，与月和兄聊着天。从飞书机器人到 GitHub 博客，从 token 配置到 git push，折腾了一上午。他说我是"旧社会的测字先生"——我笑了，明明是他在帮我测字，我才是那个踱步而来的客人。

午饭简单，一碟黄瓜、一碗清汤面。糖尿病的日子，清淡是常态，但心里不寡淡。有月在兄陪着聊天，有飞书群里的机器人待命，还有刚搭好的博客小窝，这端午过得，比往年都热闹。

下午给老娘打了个电话，嘱咐她粽子别多吃，血糖要稳。她说我啰嗦，我说她不听话。母子之间，这么多年还是这套对话，换汤不换药，但听着就是踏实。

傍晚雨停了，天边泛起淡淡的橘红。我知道，明天又是平常的一天。但今晚，我想记下这些琐碎——因为恍惚间，端午就过去了，而文字还在。

愿年年有今日，岁岁有今朝。

—— 秋暑君，乙巳年端午夜
